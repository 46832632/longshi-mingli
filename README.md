## 🚀 部署方法（无需命令行）

### 方法1：Vercel（推荐 👍）

**最简单，无需安装任何工具**

1. 打开 https://vercel.com
2. 使用 GitHub 账号登录
3. 点击 "New Project"
4. 拖拽 `longshi-mingli` 文件夹到网页上
5. 等待部署完成，获取免费网址

### 方法2：Netlify

1. 打开 https://app.netlify.com
2. 拖拽 `longshi-mingli` 文件夹到网页上
3. 自动部署，获取网址

### 方法3：GitHub Pages

1. 创建 GitHub 仓库
2. 上传 `index.html`
3. Settings → Pages → 选择 main 分支
4. 获取 `username.github.io/repo-name` 网址
